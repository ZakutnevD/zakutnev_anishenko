# Маджонг Соедини Пары

Классическая игра "Маджонг", где нужно находить пары свободных фишек и соединять их линией с не более чем двумя поворотами.

## 🎮 Как играть

1. Кликните на первую фишку — она подсветится
2. Кликните на вторую фишку — если это пара и их можно соединить линией с ≤ 2 поворотами, фишки исчезнут
3. Цель — убрать все фишки с поля

## 🚀 Запуск

Откройте файл `index.html` в любом современном браузере.

## 🛠️ Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6)

## 🧩 Особенности

- 3 уровня сложности: 4x4, 6x6, 8x8
- Анимация появления фишек
- Автоматическая подсветка возможных пар
- Перемешивание поля
- Подсказки
